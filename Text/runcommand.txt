🚀 1. Clone the Project

Open terminal (VS Code terminal recommended):

git clone https://github.com/JadhavAnsh/Formata.git
cd Formata
🧠 2. Backend Setup (FastAPI)
👉 Step 1: Go to backend
cd backend
👉 Step 2: Create virtual environment (VERY IMPORTANT)
python -m venv venv
👉 Step 3: Activate it

Windows:

venv\Scripts\activate

Mac/Linux:

source venv/bin/activate
👉 Step 4: Install dependencies
pip install -r requirements.txt
👉 Step 5: Create .env file

Create file manually OR run:

echo API_KEY=123456 > .env
echo STORAGE_PATH=./storage >> .env
echo MAX_FILE_SIZE=100 >> .env

(Optional OpenAI)

echo OPENAI_API_KEY=your_key >> .env
👉 Step 6: Run backend server
uvicorn app.main:app --reload

✅ Backend running at:

http://localhost:8000
🎨 3. Frontend Setup (Next.js)
👉 Step 1: Go to frontend

Open new terminal:

cd Formata/frontend
👉 Step 2: Install packages
npm install
👉 Step 3: Create .env.local
echo NEXT_PUBLIC_API_URL=http://localhost:8000 > .env.local
echo NEXT_PUBLIC_API_KEY=123456 >> .env.local
👉 Step 4: Run frontend
npm run dev

✅ Frontend running at:

http://localhost:3000
🔥 4. Test if Everything Works

Open browser:

UI → http://localhost:3000
API docs → http://localhost:8000/docs

Try uploading a file ✔️

✏️ 5. How to Make Changes (IMPORTANT)
🧩 Frontend Changes

Go to:

frontend/components/
frontend/app/

Example:

Change UI text → edit .tsx files
Add button → components/ui
API calls → frontend/services/

👉 Save file → auto reload

⚙️ Backend Changes

Go to:

backend/app/

Example:

API route → api/
Logic → services/
Add new feature → create new file in services

👉 Server auto reloads (because of --reload)

💡 6. Common Errors + Fix
❌ Error: pip not found
python -m pip install -r requirements.txt
❌ Error: node not found

Install Node.js (v18+)

❌ Port already in use

Change port:

uvicorn app.main:app --reload --port 8001
